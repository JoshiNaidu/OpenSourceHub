# Contributing to OpenSourceHub

Thank you for contributing to OpenSourceHub! 🚀

Our goal is to build the world's most useful directory of open-source software.

## How to Contribute

### Add a New Project

1. Fork the repository.
2. Choose the appropriate category file in `/data`.
3. Add the project using the required format.
4. Submit a Pull Request.

### Project Requirements

Projects should:

* Be open source
* Have an active repository
* Include clear documentation
* Provide real value to users
* Fit the selected category

## Project Schema

```json
{
  "name": "Project Name",
  "github": "https://github.com/owner/repository",
  "description": "Short description",
  "category": "AI",
  "tags": ["tag1", "tag2"]
}
```

## Pull Request Checklist

* [ ] Project repository is public
* [ ] Description is accurate
* [ ] Category is correct
* [ ] JSON format is valid

Thank you for helping improve OpenSourceHub.
