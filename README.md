# 🚀 OpenSourceHub

> The Dictionary of Open Source.

Discover the best open-source software for every category.

OpenSourceHub is a community-driven directory that helps developers, founders, students, and businesses discover production-ready open-source projects without spending hours searching GitHub.

From AI and Developer Tools to Databases, Authentication, CMS, DevOps, Self-Hosted Apps, and more — everything is organized in one place.

## 📌 Project Format

Each project entry contains:

```json
{
  "name": "Project Name",
  "github": "https://github.com/owner/repo",
  "description": "What the project does",
  "category": "AI",
  "tags": ["llm", "local-ai"]
}
```

## 🤝 Contributing

Want to add a project?

1. Fork this repository
2. Add a project entry to the appropriate category
3. Create a Pull Request
4. Wait for review

Please ensure:

* The repository is public
* The project is actively maintained
* The description is clear and accurate
* The project fits the selected category
