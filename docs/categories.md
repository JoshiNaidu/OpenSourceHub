# Categories

OpenSourceHub organizes projects into the following categories:

## 🤖 AI

Large Language Models, AI Agents, AI Frameworks, RAG, Vector Databases, Local AI.

## 💻 Developer Tools

CLI tools, IDE extensions, API tools, testing tools, productivity tools.

## 🔐 Authentication

Identity providers, OAuth solutions, authentication services.

## 🗄️ Databases

SQL, NoSQL, Vector Databases, Caching Systems.

## 🏠 Self-Hosted

Applications and services that can be hosted independently.

Future categories may be added based on community demand.
