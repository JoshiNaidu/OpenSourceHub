# Submission Guidelines

Thank you for contributing to OpenSourceHub.

## Requirements

Projects must:

* Be open source.
* Have a public repository.
* Include documentation.
* Be actively maintained.
* Provide real value to users.

## Project Format

```json
{
  "name": "Project Name",
  "github": "https://github.com/owner/repository",
  "website": "https://example.com",
  "description": "Short description",
  "category": "AI",
  "tags": ["tag1", "tag2"],
  "license": "MIT",
  "language": "Go",
  "self_hosted": true
}
```

## Pull Request Rules

* Add projects to the correct category.
* Avoid duplicates.
* Keep descriptions concise.
* Verify repository links before submission.
